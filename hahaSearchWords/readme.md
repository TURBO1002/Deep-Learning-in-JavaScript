### 欢迎使用hahaSearchWords

```hahaSearchWords```基于[金山词霸](http://www.iciba.com/word?w=)提供的```API```接口，帮助用户及时查询网页中不认识的单词。

### 安装

注：本应用没有发布在```Chrome```网上应用店中，用户可以将该代码下载至本地，并在本地完成加载。

<div align=center><img src="https://img-blog.csdnimg.cn/20210712094823948.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3dlaXhpbl80MTc2NzgwMg==,size_16,color_FFFFFF,t_70"/></div>

### 使用方法
  + 用户选中网页上要查询的内容，单击鼠标右键，即可在```contextmenu```中看到「金山词霸：要查询的内容」选项，单击该选项即可在新打开的页面中得到查询结果。
